颜色选择器

我们可以给任何的标签元素添加data-toggle="tooltip"属性来实现静态提示框

同时我们可以添加更过属性来实现不同效果，具体属性有以下：

  1. data-container="body"：弹出框添加到哪个元素标签内，如果不设置，默认添加在body中；可以设置元素的id格式为#id形式

  2. data-placement="left"：方向，表示提示信息出现的位置，可以是：top，bottom，left，right，auto

  3. data-delay="100"：延迟出现时间，毫秒数

  4. data-trigger="hover"：表示触发什么事件显示提示：hover鼠标经过（默认），click单击，focus得到焦点（只能应用在可以得到焦点的元素上，比如：<a>，<button>，<input>，<textarea>等），这里不建议修改

默认：鼠标经过显示，离开隐藏
