表格嵌套

我们可以在表格中嵌套表格

我们同时也可以给单元格添加对齐方式类.text-left居左，.text-center居中，.text-right居右，.vertical-top上对齐（默认），.vertical-middle中对齐，.vertical-bottom底对齐
