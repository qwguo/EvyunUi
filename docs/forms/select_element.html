下拉菜单

我们在做表单下拉菜单的时候，给表单的class="select-element"元素添加dropdown类，并且给class="input-element"子类添加下列属性来实现js交互；

  1. data-type="select"：表示这个是一个表单下拉选择菜单，固定格式
  2. data-toggle="dropdown"：用来表示调用js事件，固定格式
  3. data-multiple="0"：用来表示是否支持多选：0表示否，1表示是
