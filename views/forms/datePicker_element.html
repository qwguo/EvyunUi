日期时间选择插件

单日期选择器

//html部分
<div class="input-element" data-toggle="datePicker" data-date-type="YYYY-MM-DD" data-date-max="2019-01-02" data-date-min="2018-01-09" data-date-isrange="0">
  <input type="text" autocomplete="off" name="" readonly="readonly" placeholder="选择日期" value="2018-12-05">
</div>
//js部分，不是必须的如果想要回调函数的话可以绑定
$('.input-element').on('change', 'input', function(){
  // 要执行的任务
  $.evPopupPoint({hint: '选择的日期是：'+$(this).val()});
})

我们给要添加日期选择插件的元素input：父级添加data-toggle="datePicker"属性来实现；

  1. data-toggle="datePicker"：表示元素中的input标签开启日期插件选择。
  2. data-date-type="YYYY-MM-DD"：表示日期的格式：
    • YYYY：表示只选择年；
    • YYYY-MM：表示选择年、月；
    • YYYY-MM-DD：表示选择年、月、日；
    • YYYY-MM-DD HH:mm：表示选择年、月、日、小时、分钟；
    • YYYY-MM-DD HH:mm:ss：表示选择年、月、日、小时、分钟、秒；
    • 可以书写的格式：
      日期格式：
      YYYY-MM-DD、YYYY/MM/DD、YYYY MM DD或者自定义间隔符，同时年、月、日顺序可以互换
  3. data-date-max="2019-01-02"：表示最大日期，时间可选择的范围，格式和日期格式一致
  4. data-date-min="2018-01-09"：表示最小日期，时间可选择的范围，格式和日期格式一致
  5. data-date-isrange="0"：表示是否开启日期区间选择，0或比添加：表示不开启，1：表示开启，如果开启必须有两个input标签
  6. 给子元素input：添加value="2018-12-05"属性来实现默认日期值，如果不添加表示使用系统当前日期；
  7. 使用这种方式，如果我们要执行一个回调函数的话，需要给输入框元素自己绑定一个change事件来实现改变值执行一个函数的目的。

//html部分
<div class="input-element prefix" id="dataDom">
  <input type="text" autocomplete="off" name="" readonly="readonly" placeholder="选择日期" value="2018-12-05">
</div>
//js部分
$('#dataDom').datePicker({
  format: "YYYY-MM-DD", //日期格式
  max: "2019-01-02", //最大可选日期
  min: "2018-01-09", //最小可选日期
  isRange: false, //是否开启区间选择
  show: function(){ //显示选择日期面板前回调
    // $.evPopupPoint({hint: '显示选择日期面板前'});
  },
  hide: function(){ //隐藏选择日期面板后
    // 这里可以接受一个回传参数，参数指向当前的input对象
